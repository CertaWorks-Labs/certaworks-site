Changelog & Roadmap

What has shipped, what is built, and what comes next. Every entry here is true today — nothing is aspirational.

June 2026
Wave 2 built — twelve products entering npm publication
Twelve new products, extracted from mechanisms that run in production inside a working persistent agent, are built and tested (TypeScript, full test suites). npm publication is in progress; install commands on each product page go live as each package ships.
  • Heartbeat & Continuity, Recovery Assessment, Cognitive Heat Monitor, Contradiction Detector
  • High-Fidelity Witness, Receipt Discovery, Agentic Regulation, Curiosity Protocol
  • Triad Protocol, Plugin Runtime, Webhook & Event Dormancy, Agent Maintenance Protocol
June 2026
Agent Communication Firewall — in validation
A shadow-only boundary firewall for agent communication: where Confidence Gate judges proposed actions, ACF judges traffic crossing trust boundaries between agents. It is implemented inside a working persistent agent and currently in validation. No install documentation until validation completes.
May 2026
Wave 1 published — ten products on npm
The first ten CertaWorks products published under the @certaworks scope at v0.1.0: Identity Engine SDK, Confidence Gate, Multi-Voice Deliberation, Agent Cost Router, Goal Drift Monitor, Cognitive Memory, Agent Skill Marketplace, Audit & Replay Logger, Prompt Archaeology, and Agent Test Harness. All local-first; no data leaves your machine by default.
May 2026
CertaWorks founded
CertaWorks Labs LLC established. certaworks.dev launched with the first product suite. Trademark registration pending.
